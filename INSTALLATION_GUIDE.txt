Q: Is my data safe?
A: Yes! Data is stored locally in your browser. Nobody else can 
   access it. But remember to backup regularly.

Q: Can I use this on multiple computers?
A: Yes! Just copy the kingg-ledger folder to other computers and 
   install Node.js there. But data won't sync automatically - you 
   need to manually transfer backups.

Q: Can multiple people use it at once?
A: Not simultaneously on same computer. But you can run it on 
   multiple computers with separate data.

Q: What if I forget the password?
A: Use Developer Tools (F12) â†’ Console â†’ Type:
   localStorage.clear()
   This will reset everything including password to default.

Q: Can I access this from my phone?
A: Not directly. This runs on your computer only. To access from 
   phone, you'd need to deploy it to a web server (advanced).

Q: Is internet required?
A: Only for initial installation (to download packages). After 
   that, it works completely offline!

Q: How do I update the app?
A: Download new version, extract, copy your data backup, run new 
   version, restore backup.
